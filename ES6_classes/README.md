# ES6 Classes

This project implements class-based exercises using ES6 syntax and features.
It includes all required tasks from 0 to 10.

## Project Information

- Repository: holbertonschool-web_back_end
- Directory: ES6_classes
- Language: JavaScript (ES6)

## Learning Objectives

- Define and use ES6 classes
- Add methods, getters, and setters
- Validate constructor input types
- Use inheritance and method overriding
- Model abstract-like base classes
- Use symbols and primitive conversion hooks
- Fix hoisting and class initialization issues

## Files

- 0-classroom.js: ClassRoom class with max student size
- 1-make_classrooms.js: initializeRooms function creating 3 classrooms
- 2-hbtn_course.js: HolbertonCourse class with validated setters/getters
- 3-currency.js: Currency class with displayFullCurrency
- 4-pricing.js: Pricing class with full price display and converter
- 5-building.js: Building abstract-style class contract
- 6-sky_high.js: SkyHighBuilding subclass with overridden warning message
- 7-airport.js: Airport class with custom default string tag
- 8-hbtn_class.js: HolbertonClass with Number/String casting behavior
- 9-hoisting.js: Fixed hoisting implementation with class/student exports
- 10-car.js: Car class with cloneCar and Symbol.species
