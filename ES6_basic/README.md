# ES6 Basics

This project covers the fundamental ES6 features used in modern JavaScript development.
It contains solutions for tasks 0 to 12 from the Holberton ES6 Basics module.

## Project Information

- Repository: holbertonschool-web_back_end
- Directory: ES6_basic
- Language: JavaScript (ES6)
- Runtime: Node.js 20.x

## Learning Objectives

By completing this project, you practice:

- Using `const` and `let` correctly
- Understanding block scope
- Writing arrow functions
- Using default parameters
- Using rest and spread syntax
- Working with template literals
- Using object property shorthand
- Using computed property names
- Using ES6 method properties
- Iterating with `for...of`
- Building objects dynamically

## Files

- 0-constants.js: Replace `var` with `const` and `let`
- 1-block-scoped.js: Prevent variable overwrite with block scoping
- 2-arrow.js: Convert function to arrow function syntax
- 3-default-parameter.js: Use default parameter values
- 4-rest-parameter.js: Count arguments using rest parameters
- 5-spread-operator.js: Concatenate arrays and string characters with spread
- 6-string-interpolation.js: Build output using template literals
- 7-getBudgetObject.js: Use object property value shorthand
- 8-getBudgetCurrentYear.js: Use computed property names
- 9-getFullBudget.js: Use ES6 method property syntax
- 10-loops.js: Rewrite loop using `for...of`
- 11-createEmployeesObject.js: Create department-employees object
- 12-createReportObject.js: Create report object with method

## Requirements

- Ubuntu 20.04 LTS (or compatible environment)
- Node.js 20.x
- npm 10.x or later

## Setup

1. Install Node.js 20.x.
2. Install project dependencies in the project root:

```bash
npm install
```

If needed for this module, install Babel, ESLint, and Jest in your setup:

```bash
npm install --save-dev @babel/core @babel/preset-env @babel/cli
npm install --save-dev eslint
npm install --save-dev jest
```

## Run Examples

Use the provided main files (for example: `0-main.js`, `1-main.js`, etc.):

```bash
npm run dev 0-main.js
npm run dev 1-main.js
npm run dev 2-main.js
```
